# 01_python — Trilha de Python

Esta pasta concentra a trilha de Python do **portifolio_quantum**, do básico até entregáveis aplicados (fundamentos, quântica, PQC/segurança, redes e research).

Começar por:
- `00_setup/` (ambiente e padrões)
- `01_basics/` (revisão do zero)
- `02_practice/` (exercícios e repetição)

Entregáveis (projetos reproduzíveis) ficam nas fases:
- `10_fundamentos/` → `50_research/`

Documentos desta trilha:
- `ORGANIZACAO.md` — onde colocar cada coisa e como a pasta se organiza
- `PADROES.md` — nomenclatura e template padrão de projetos
- `METODOLOGIA.md` — progressão, rotina e critérios de avanço
