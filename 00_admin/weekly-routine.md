# Weekly Routine

Modelo semanal simples (repete toda semana).

## Sessões
- 2x Python
- 1–2x Matemática (sempre com exercícios)
- 1x Quântica (Katas/teoria)
- 1x Quântica (Qiskit/prática)

## Regra do entregável semanal
Toda semana deve terminar com pelo menos 2 itens abaixo:
- 1 arquivo de notas (markdown)
- 1 lista de exercícios resolvida (mesmo parcial)
- 1 notebook Qiskit atualizado
- 1 mini-projeto Python com commit
